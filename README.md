## Analisi dei Dati e Big Data — Progetto d'esame

Progetto d'esame del corso di Analisi dei Dati e Big Data (Prof. Marco Del Coco),
Data Science per le Scienze Umane e Sociali — Università del Salento.

Il progetto esplora tre paradigmi di gestione dati diversi, applicati a tre casi
d'uso realistici:


Studio Medico — database relazionale (SQLite): gestione di pazienti,
medici, visita e prescrizioni.
Grafo Shop — database a grafo (Neo4j): e-commerce con campagne
basatevi sulle relazioni tra clienti e prodotti.
Piattaforma Streaming — database documentale (Elasticsearch): catalogo di
film, serie TV e documentari con ricerca testo completo e aggregazioni.


Ogni cartuccia contiene il proprio codice, i dati e un README dedicato con le
istruzioni specifiche per eseguire quella traccia.
